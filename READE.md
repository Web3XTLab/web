# DApp Store

## Need To Know

- [Truffle](https://trufflesuite.com/tutorial/)
- [Ganache](https://trufflesuite.com/ganache//)

## Install

```bash
$ npm install -g truffle

$ npm run install
```

## Compile & Migrate Contract
```bash
$ npm run compile

$ npm run migrate
```

## Dev

```bash
npm run dev
```
